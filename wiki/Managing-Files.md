# Managing Files

> _This feature is planned. This page will be updated when the file browser ships._

---

## Overview

InkLink includes a file browser that lets you navigate the filesystem on your CrossPoint reader. From the browser you can rename, move, and delete files without touching the reader itself.

Under the hood, InkLink uses **WebDAV**, a standard file-access protocol that CrossPoint exposes in Sync mode.

## Tips

- Your reader must be in **Sync mode**. See [Getting Started](Getting-Started.md).
- Avoid modifying files that CrossPoint uses for its own state (bookmarks, settings). Editing these by hand can cause unexpected behavior on the reader.
- If an operation fails, see [Troubleshooting](Troubleshooting.md).
