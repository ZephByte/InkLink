# Fonts

> _This feature is planned. This page will be updated when font management ships._

---

## Overview

CrossPoint supports custom fonts via `.inx` font packs. InkLink lets you install them directly from your phone without using the web UI or a USB cable.

---

## What Is a Font Pack?

An `.inx` file is the font pack format used by CrossPoint firmware. It bundles one or more font files in the format the reader expects.

_More details on the format will be documented here once confirmed against the firmware._

---

## How to Install a Font

_Steps will be documented once the feature ships._

The planned flow:
1. Get a `.inx` font pack file onto your phone (download, AirDrop, share sheet, etc.)
2. Open InkLink and go to **Fonts**
3. Tap **Install Font**
4. Pick the `.inx` file
5. InkLink uploads it to the correct location on your reader
6. The font becomes available in CrossPoint's font settings

## Tips

- Your reader must be in **Sync mode**. See [Getting Started](Getting-Started.md).
- Font packs take effect the next time you open a book or change the font setting on the reader. You may need to restart the reader's reading app.
